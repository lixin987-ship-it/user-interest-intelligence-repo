# Sample User Data

This folder is intended to store user behavior data for analysis. 

## Data Structure Examples

### Daily User Behavior Format
```
UserId | Anid | Muid | Date | DetailedSource | Source | Action
```

### Historical User Interest Profile Format  
```
User Id | Market | From Date | To Date | Category | Sub Category | Keywords | Source | Weight | Duration
```

## Privacy Notice

When uploading data to this folder:
- Ensure all data is properly anonymized
- Remove any personally identifiable information (PII)
- Use hashed or pseudonymized user identifiers
- Comply with all applicable data protection regulations

## File Naming Convention

Use descriptive filenames such as:
- `daily-behavior-YYYY-MM-DD.csv`
- `historical-profile-YYYY-MM-DD.csv`
- `sample-dataset-description.txt`
